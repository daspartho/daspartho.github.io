generative ai

  • i built prompt extend to help with prompt engineering for stable diffusion models by automatically generating suitable style cues to add to the prompt and it's fully open-source with training code and model weights available! also received 2x $1000 grant from algovera.ai for the project.
  • i'm currently working on DataGPT to easily generate synthetic data using LLMs for distilling LLM knowledge into much smaller and faster-to-run models.
  • i previously actively contributed to the huggingface transformers and diffusers library and currently contributing to the LLM-VM project.
  • i implemented the MagicMix: semantic mixing with diffusion models paper, DiffEdit: diffusion-based semantic image editing with mask guidance paper, and some other simple implementations of popular transformer papers i wrote.
  • some other ML projects i made
    • predict subreddit: an NLP model for predicting subreddit based on the title of a post
    • bored ape diffusion: a diffusion model for unconditional image generation of bored apes
    • is it huggable: an image classifier that determines whether something is huggable or not
    • anime or not: an NLP model that tells if a plot is anime enough
